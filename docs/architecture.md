# Архитектура фронтенда (Frontend Architecture)

Фронтенд Poputki.online реализован как современное одностраничное приложение (SPA) на базе Vue 3.

## Стек технологий

- **Vue.js 3**: Фреймворк для построения интерфейса.
- **Vite**: Инструмент сборки и сервер разработки.
- **Tailwind CSS**: Фреймворк для стилизации (Vanilla CSS + Tailwind).
- **Vue Router**: Управление навигацией.
- **Axios**: HTTP-клиент для взаимодействия с API.
- **Telegram WebApp SDK**: Интеграция с функциями Telegram (цвета, кнопка «Назад», тактильная отдача).

## Структура проекта

- `src/`: Основной исходный код.
  - `components/`: Общие UI компоненты (кнопки, шапка, логотип).
  - `views/`: Основные страницы приложения:
    - `HomeView.vue`: Главная страница со списком поездок.
    - `RideDetailsView.vue`: Информации о конкретной поездке.
    - `ProfileView.vue`: Личный кабинет пользователя.
    - `AdminView.vue`: Панель главного администратора.
    - `BusAdminView.vue`: Панель для операторов автобусов.
  - `router/`: Конфигурация маршрутов.
  - `api/`: Настройка Axios и базового URL API.
  - `assets/`: Статические ресурсы (изображения, стили).

## Основные принципы дизайна

1. **Premium Dark UI**: Использование глубоких темных оттенков (Slate 900) и ярких акцентов (Amber 500).
2. **Адаптивность**: Интерфейс оптимизирован под мобильные устройства (Telegram Mini App).
3. **Skeleton Loaders**: Использование анимированных заглушек во время загрузки данных для плавности интерфейса.
4. **Интеграция с ТГ**: Цветовая схема приложения динамически подстраивается под настройки темы пользователя в Telegram.

## Сборка и развертывание

Приложение собирается в статические файлы командой `npm run build`. Результат находится в директории `dist/` и может быть развернут на любом статическом хостинге (Vercel, Netlify, Nginx).
