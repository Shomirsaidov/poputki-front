# Библиотека компонентов (UI Components)

В разработке фронтенда используется набор переиспользуемых компонентов для обеспечения консистентности дизайна.

## Основные компоненты

### `AppLogo.vue`
- Отображает логотип платформы (иконка + текст).
- Поддерживает различные размеры и варианты отображения (только иконка, светлый/темный текст).

### `RideCard.vue`
- Карточка поездки для главного списка.
- Отображает маршрут, время, цену, количество мест и аватар водителя.
- Использует разные стили для легковых авто и автобусов.

### `SkeletonLoader.vue` (и встроенные паттерны)
- Компоненты для визуализации загрузки.
- Реализованы через Tailwind класс `animate-pulse`.
- Скелетоны подготовлены для:
  - Карточек поездок.
  - Таблиц (манифест пассажиров).
  - Графиков в админ-панели.

### `BusSeatMap.vue`
- Интерактивная схема мест в автобусе.
- Поддерживает разные типы автобусов (одноэтажные и двухэтажные).
- Управляет состоянием выбора мест (забронировано, выбрано, пусто).

## Стилизация
- Дизайн следует принципам **Glassmorphism** (полупрозрачные карточки с размытием фона).
- Цветовая палитра:
  - Основной фон: `bg-slate-900` (в темной теме).
  - Акценты: `text-amber-500`, `bg-amber-500`.
  - Границы: `border-white/10`.

## Иконки
- Используются SVG иконки для максимальной четкости на всех устройствах.
- Источник иконок: Heroicons / Lucide.
