# Справочник страниц и маршрутов (Views and Routing)

Приложение Poputki.online разделено на несколько функциональных зон, доступных по разным маршрутам.

## Основные маршруты

### Поиск и Главная (`/`)
- Файл: `HomeView.vue`.
- Функционал: Поиск поездок, фильтрация по городам, отображение списка доступных вариантов. Содержит переключатель между легковыми авто и автобусами.

### Детали поездки (`/ride/:id`)
- Файл: `RideDetailsView.vue`.
- Функционал: Полная информация о рейсе, детали водителя/машины, кнопка бронирования. Для автобусов отображается интерактивная схема мест.

### Личный кабинет (`/profile`)
- Файл: `ProfileView.vue`.
- Функционал: Редактирование профиля (имя, машина, телефон), список моих поездок и бронирований.

### Создание поездки (`/create`)
- Файл: `CreateRideView.vue`.
- Функционал: Форма создания нового рейса для водителей или заявки для пассажиров. Поддержка обратных рейсов.

## Специализированные панели

### Панель администратора (`/admin`)
- Файл: `AdminView.vue`.
- Функционал: Глобальная статистика (дашборд с графиками), управление пользователями, рейсами и городами. Доступ защищен пин-кодом.

### Панель оператора автобусов (`/bus-admin`)
- Файл: `BusAdminView.vue`.
- Функционал: Создание автобусных рейсов, просмотр манифеста пассажиров, управление бронированиями. Вход по телефону и паролю.

### Бронирование автобуса (`/bus-booking/:id`)
- Файл: `BusBookingView.vue`.
- Функционал: Специализированная форма бронирования для автобусов с заполнением паспортных данных.

## Навигация
- Приложение использует `vue-router`.
- Большинство переходов осуществляется через `this.$router.push()`.
- Интегрирована кнопка «Назад» от Telegram SDK для плавного UX внутри мессенджера.
